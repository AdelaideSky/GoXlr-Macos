## Set up

- Download the latest [release](https://github.com/AdelaideSky/GoXlr-Macos/releases/latest).
- Open the .zip file and put the app file into your Applications folder
- Open the app and follow the onboarding process displayed !

## Notice

- The app provides an alternative UI more integrated within the system for the GoXLR-Utility, as well as audio support and plug-and-play. The app also features original functionalities such as run shortcut on button press or run command on system sleep.
- The bridge between the daemon and Swift is handled by the GoXlrKit.
